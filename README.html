Comment mettre à jour le programme de l'Arduino

Étape 1:

Débrancher le câble d'alimentation de la boîte.

Ouvrir la boîte et connecter l'arduino à l'ordinateur avec le cable mini USB.

Étape 2:

Télécharger et installer le logiciel Arduino.

Ouvrir le ficher "LED-strip-IR-controler.ino" avec le logiciel Arduino.

Cliquer sur "Croquis" -> "Inclure une bibliothèque" -> "Gérer les bibliothèques".

Rechercher la librairie IRremote (par Armin Joachimsmeyer) et l'installer.

Cliquer sur fermer.

Étape 3:

Cliquer sur "Outils" -> "Type de carte" et selectionner "Arduino Nano"

Cliquer sur "Outils" -> "Port" et selectionner l'option disponible.

Si aucune option n'est disponible, débrancher rebrancher l'arduino, relancer le logiciel, redémarer l'ordinateur. Si le problème perciste, me contacter.

Si plus d'une option est disponible, me contacter.

Étape 4:

Cliquer sur le bouton bleu ✔ (surnommé Vérifier).

Si la fernêtre affiche "Compilation terminée" sur un fond bleu, passer à l'étape 5.

Sinon (erreur sur fond rouge), me contacter.

Étape 5:

Cliquer sur le bouton ➡ (surnommé Téléverser).

Si la fenêtre affiche "Téléversement terminé" sur fond bleu, c'est terminé ! Déconnecter l'arduino de l'ordinateur et refermer le boîte.

Sinon, me contacter.
